# A.M.V.A. Application

## What is A.M.V.A.?
A.M.V.A are the initials of the Metropolitan Area of the Aburrá Valley ("Área Metropolitana del Valle de Aburrá") in Spanish.

## Pre-requisites
- [Node.JS](https://nodejs.org/)
